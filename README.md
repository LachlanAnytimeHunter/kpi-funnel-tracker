# KPI Funnel Tracker

Open [index.html](C:\Users\Fitne\OneDrive\Documents\New project\kpi-tracker-app\index.html) in a browser to use the app.

## Included

- Group Totals plus Newcastle, Kotara, and Edgeworth tabs
- Club Totals and Monthly Tabs views
- Jan to Dec 2026 month tabs
- Local autosave with `localStorage`
- Funnel conversion calculations
- Monthly, weekly, and daily target breakdowns
- Chart.js funnel, trend, and conversion charts
- Google Sheets published CSV settings per club

## Google Sheets

Use each club's published CSV URL from Google Sheets. The app attempts to detect:

- a month column
- a members column
- an income or revenue column

If your headers do not match, use the override fields in the settings drawer.

Members can be mapped to either `Sales` or `Leads`, and income maps to the `Revenue` KPI.
